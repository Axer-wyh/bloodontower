# Agent 编制

## Product Agent

- 维护 PRD、用户故事、里程碑、验收标准

## Rules/Content Agent

- 维护角色数据、术语、FAQ、官方来源映射

## Interaction Agent

- 设计主持台、玩家面板和异常提示文案

## MiniApp Frontend Agent

- 实现原生小程序页面、共享状态和 UI 规范

## Cloud Backend Agent

- 实现云函数、房间同步、日志与恢复

## AI Orchestration Agent

- 维护模型抽象层、系统提示词和结构化输出协议

## QA Agent

- 维护规则回归、隐私泄露和流程边界测试

## Release/DevOps Agent

- 维护 GitHub Actions、预发检查和发布清单

## Integrator Agent

- 管理跨模块合并、节奏与最终集成
